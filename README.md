# Discord Data Bot

> On-demand data access for teams. One message, one answer. Democratized analytics across the company.

## Problem

As the data person on the team, I was constantly interrupted with information requests. The team needed access to analytics but couldn't query databases directly.

## Solution

Built a Discord bot that interprets natural language commands and returns structured data. The team can self-serve analytics 24/7 without waiting for me.

## Impact

- Freed up hours of weekly interruptions
- Democratized data access across the organization
- Available to the entire team, any time

## Tech Stack

- Python
- Discord API
- Data processing and formatting
